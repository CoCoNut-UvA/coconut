===============
Primitive types
===============

CoConut provides some primitive types to be used in attributes and trav data.
The table below shows the types in CoCoNut and their corresponding C type.

+------------------+----------------+
| CoCoNut type     |  C type        |
+==================+================+
| int              | int            |
+------------------+----------------+
| int8             | int8_t         |
+------------------+----------------+
| int16            | int16_t        |
+------------------+----------------+
| int32            | int32_t        |
+------------------+----------------+
| int64            | int64_t        |
+------------------+----------------+
| uint8            | uint8_t        |
+------------------+----------------+
| uint16           | uint16_t       |
+------------------+----------------+
| uint32           | uint32_t       |
+------------------+----------------+
| uint64           | uint64_t       |
+------------------+----------------+
| bool             | bool(stdbool.h)|
+------------------+----------------+
| float            | float          |
+------------------+----------------+
| double           | double         |
+------------------+----------------+
| string           | char *         |
+------------------+----------------+
